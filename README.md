# AI PDF Chatbot

This project is a Streamlit-based AI chatbot that allows users to upload a PDF and interact with its content.

## Features

* Upload any PDF
* Extract text from the document
* Display the content inside a web interface
* Built using Python and Streamlit

## Technologies Used

* Python
* Streamlit
* PyPDF
* LangChain
* FAISS

## Installation

Clone the repository:

git clone https://github.com/rishitha18-git/ai-pdf-chatbot.git

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
